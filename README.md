# lap2_ARTI-308

## Dataset
An open Titanic dataset was selected from Kaggle. The dataset contains passenger information such as age, gender, ticket class, fare, and survival status.

## Machine Learning Problem
This project addresses a supervised classification problem. The objective is to predict whether a passenger survived the Titanic disaster based on available features. The target variable is survival.

## Tools Used
- Kaggle Notebook
- Python
- Pandas

## Files Included
- Jupyter Notebook for loading and inspecting the dataset
- Methodology diagram illustrating the ML workflow
